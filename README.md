# Code of Conduct:

# Overview:
# The Megradea Software Project values respect and inclusivity, expecting all participants to be courteous and respectful in all interactions. This Code of Conduct applies to all project spaces, including online communications and in-person events.;

# Contact for Reporting;
    
# Report concerns to paulwandendeya@gmail.com. Reports are confidential and may involve other team members.;

# Details:
    
# The Megradea Software project ensures a welcoming environment for all. We don't tolerate disrespectful behavior, discrimination, or harassment based on background, identity, or any characteristic. This Code applies to all project spaces. By participating, you agree to follow this Code.;

# Expected Behaviour
# Be respectful, inclusive, and considerate.Avoid exclusionary behavior, harassment, or discomfort. Be mindful of language and impact on others. Respect boundaries and consent. Speak up if you see issues. Be professional and humble in interactions.;

# Unacceptable Behaviour
# Includes: harassment, threats, violence, exclusionary comments, disruption, and non-consensual interactions. This includes verbal, written, or physical actions that cause discomfort or fear.;

# Consequences of Unacceptable Behaviour;
# Stop unacceptable behavior when asked. Violations may lead to warnings, removal from events/platforms, or other sanctions.;
